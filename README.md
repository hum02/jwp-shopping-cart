## 상품 페이지 연동
1. 도메인 product 구현하기
   - [x] 상품 기본 정보 필드 원시값으로 저장하기 
     - 상품 ID
     - 상품 이름
     - 상품 이미지
     - 상품 가격
  
2. 데이터 베이스에 저장된, 상품 조회하기 기능 구현
      - [x] 데이터 베이스 테이블 생성
      - [x] 레포지토리 구현
      - [x] 서비스 구현
      - [x] 컨트롤러 구현
